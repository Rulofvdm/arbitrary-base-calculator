# Base Division Calculator

A small static site that divides two numbers in any base (2–36) and shows the **quotient** and **remainder**.

## Try it locally

Open `index.html` in a browser, or serve the folder:

```bash
# Python
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Host on GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, choose **Deploy from a branch**.
4. Select branch `main` (or `master`) and folder **/ (root)**.
5. Save. Your site will be at `https://<username>.github.io/<repo-name>/`.

No build step required—just static HTML, CSS, and JavaScript.
